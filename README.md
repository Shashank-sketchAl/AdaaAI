## Project Title
AdaaAI — AI Stylist Agent for Lucknow Ethnic Wear Brands

## Problem Statement
PS-17 — AI stylist agent for Lucknow ethnic wear brands

Lucknow has a booming ethnic fashion market including chikankari, zardozi, kurtas, sarees, and festive wear, but most brands lack personalization and AI-powered shopping assistance.

AdaaAI acts as a conversational AI stylist that recommends outfits based on occasion, budget, color preferences, and fashion style.

---

## Planned Features
- AI-powered fashion stylist chat
- Occasion-based outfit recommendations
- Personalized ethnic styling suggestions
- Budget-aware recommendations
- Fashion accessory suggestions
- Responsive modern UI
